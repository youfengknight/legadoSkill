# 22 笔趣阁书源使用说明
## 书源信息
- **书源名称**: 22 笔趣阁
- **书源地址**: https://m.22biqu.com
- **书源分组**: 笔趣阁
- **书源类型**: 文字小说
## 功能特性
1. ✅ 支持搜索功能
2. ✅ 支持发现页浏览（9 个分类）
3. ✅ 支持书籍详情页解析
4. ✅ 支持目录章节解析
5. ✅ 支持正文内容阅读
6. ✅ 自动清理广告和脚本
## 发现页分类
- 玄幻小说
- 武侠小说
- 都市小说
- 历史小说
- 科幻小说
- 游戏小说
- 女生小说
- 其他小说
- 完本小说
## 使用方法
1. 打开 Legado 阅读 APP
2. 进入"书源管理"
3. 点击"导入书源"
4. 选择"从文件导入"
5. 选择 22biqu.json 文件
6. 导入成功后即可使用
## 文件位置
g:\Project\阅读 SKills\legadoSkill-main\temp\22biqu\22biqu.json
## 注意事项
- 本书记使用移动端站点 (m.22biqu.com) 进行解析
- 已配置移动端 User-Agent
- 支持上一章/下一章快速跳转
- 自动过滤广告和无关内容
