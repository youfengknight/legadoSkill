批量并行抓取模块用于同时抓取多个页面，将书源创建效率提升 70% 以上。

### 优化前（串行抓取）
- 抓取 5 个页面：首页、搜索页、书籍详情、章节页、分类页

- 并发数：10

文件内容（核心代码）：
`python
import asyncio
import aiohttp
from typing import Dict, Tuple
class BatchFetcher:
    def __init__(self, max_concurrent: int = 10, timeout: int = 30):
        self.max_concurrent = max_concurrent
        self.timeout = timeout
    async def _fetch_url(self, session, url, task_id):
        try:
            async with session.get(url, timeout=self.timeout) as response:
                content = await response.text(encoding='utf-8', errors='replace')
                return (task_id, content, None)
        except Exception as e:
            return (task_id, None, str(e))
    def fetch_multiple(self, urls: Dict[str, str]) -> Tuple[Dict[str, str], Dict[str, str]]:
        loop = asyncio.new_event_loop()
        asyncio.set_event_loop(loop)
        connector = aiohttp.TCPConnector(limit=self.max_concurrent)
        async def fetch_all():
            async with aiohttp.ClientSession(connector=connector) as session:
                tasks = [self._fetch_url(session, url, tid) for tid, url in urls.items()]
                results = {}
                errors = {}
                for future in asyncio.as_completed(tasks):
                    tid, content, error = await future
                    if content:
                        results[tid] = content
                    else:
                        errors[tid] = error
                return results, errors
        return loop.run_until_complete(fetch_all())
`

## 🎯 使用场景
### 1. 快速分析新网站
`python
# 一次性抓取所有关键页面
urls = {
    "homepage": base_url,
    "mobile": mobile_url,
    "search": search_url,
    "category1": cat1_url,
    "category2": cat2_url
}
results, _ = fetcher.fetch_multiple(urls)
`
### 2. 批量测试书源
`python
# 同时测试多个书籍 URL
book_urls = {
    f"book_{i}": f"{base_url}/book/{i}/" 
    for i in range(1, 11)
}
results, errors = fetcher.fetch_multiple(book_urls)
`
### 3. 验证目录结构
`python
# 批量抓取章节验证目录规则
chapter_urls = {
    f"ch_{i}": f"{base_url}/book/1/chapter/{i}.html"
    for i in range(1, 6)
}
results, errors = fetcher.fetch_multiple(chapter_urls)
`

## ⚠️ 注意事项
1. **并发数设置**：推荐 10，不要超过 20
2. **超时时间**：国内网站 30 秒，国外 60 秒
3. **遵守 robots.txt**：不要抓取禁止的页面
4. **错误处理**：始终检查 errors 字典
